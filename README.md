# Rock-Paper-Scissors

Classic Rock-Paper-Scissors browser game.

## What this is
Simple DOM-based game to practice logic and UI manipulation.

## Features
- Player vs Computer  
- Score tracking  
- Restart option  

## Preview
Open `index.html` or run:
```bash
python -m http.server 8000
```
